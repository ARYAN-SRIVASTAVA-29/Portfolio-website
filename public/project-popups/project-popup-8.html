Whisper App

Project Info:

This user authentication web application showcases my expertise in implementing secure and efficient registration and login systems using cutting-edge technologies. Developed with Express, EJS, and MongoDB, the application allows users to register and set their passwords, with the added security of salt rounds for password encryption. The integration of Passport.js, passport-local-mongoose, and Google OAuth 2.0 enables users to easily sign in using their Google accounts, offering a seamless and convenient authentication experience.

Key Features:

Secure Registration: Users can register and set their own passwords, with password encryption using salt rounds.
Google Sign-In:Integration of Google OAuth 2.0 for easy and secure registration and sign-in via Google accounts.
Passport.js & Passport-Local-Mongoose: Robust user authentication management through Passport.js and passport-local-mongoose
Cross-Browser Compatibility:Ensures a consistent authentication experience across various browsers and platforms.
