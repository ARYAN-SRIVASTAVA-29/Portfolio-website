Blog Website

Project Info:

This blog website demonstrates my expertise in creating feature-rich and interactive web applications using both front-end and back-end technologies. Developed with Express, EJS, and MongoDB, the blog website offers a platform for users to read and compose blog posts effortlessly. The application leverages MongoDB Atlas for secure and reliable cloud-based data storage, ensuring that blog posts are safely stored and easily accessible. With a dedicated "/compose" route, users can conveniently create new blog posts and seamlessly integrate them into the website.

Key Features:

Blog Post Management:Efficient organization and display of blog posts.
Compose Feature:Dedicated "/compose" route for easy creation and publishing of new blog posts.
MongoDB Atlas Integration:Secure and reliable cloud-based data storage for blog content.
Intuitive User Interface:Clean design and user-friendly experience developed using EJS, Express, and front-end technologies.
