E-Commerce WebApplication

Project Info:

This e-commerce web application is a demonstration of my expertise in full-stack web development, with a focus on creating intuitive and functional online shopping experiences. Developed using Express, Node.js, HTML, CSS, and JavaScript, the application showcases my ability to integrate essential back-end and front-end technologies to deliver a seamless user experience.

Key Features:

Fully Functional Cart: Fully functional cart with real-time item count badge updates.
Package Utilization: Effective utilization of firebase-functions, express, body-parser, ejs, and path packages.
Smooth User Experience:Smooth user experience optimized for performance and usability.
Responsive Design: Responsive design for various devices and screen sizes.
