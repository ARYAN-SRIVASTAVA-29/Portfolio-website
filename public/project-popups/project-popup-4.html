ToDo List App

Project Info:

This to-do list application showcases my ability to create practical and efficient web applications using a combination of front-end and back-end technologies. Developed with Express, EJS, and MongoDB, the app allows users to manage their tasks effectively by organizing them based on specific days. The application leverages MongoDB Atlas for secure and reliable cloud-based data storage, while providing an intuitive interface for users to navigate between different days using routes like "/yesterday" and "/tomorrow."

Key Features:

Task Management:Efficient organization of tasks based on specific days.
Dynamic Routing:Seamless navigation between days using routes like "/yesterday" and "/tomorrow".
MongoDB Atlas Integration:Secure and reliable cloud-based data storage for tasks.
Intuitive User Interface:Clean design and user-friendly experience developed using EJS, Express, and front-end technologies.
