Weather Application

Project Info:

This weather application is a testament to my proficiency in front-end web development, showcasing the ability to create interactive and dynamic web applications using HTML, CSS, and JavaScript. Leveraging the power of a weather API, the application allows users to access real-time weather data for any city around the world, displaying essential weather details like temperature, humidity, and wind conditions. The application also enhances the user experience by changing the background image according to the current time and weather conditions of the chosen city.

Key Features:

Global Weather Access:Real-time weather data for any city around the world.
Comprehensive Weather Details: Displays temperature, humidity, wind conditions, and weather status (e.g., cloudy)
Dynamic Background:Background image changes based on the time and weather conditions of the selected city
User-friendly Interface:Clean and intuitive design, developed using HTML, CSS, and JavaScript.
